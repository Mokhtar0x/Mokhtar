## 🔄 إعادة التعيين السريعة

### إعادة تعيين كاملة:
```bash

```
هذا سيقوم بـ:
1. إيقاف البوت
2. نسخ احتياطي للبيانات
3. حذف قاعدة البيانات
4. مسح السجلات
5. بدء البوت من جديد

## ❗ حل المشاكل السريع

### البوت لا يرد؟
```bash
# أوقف البوت
taskkill /F /IM node.exe

# انتظر 3 ثواني
timeout /t 3

# ابدأ البوت
start-bot.bat
```

### خطأ 409 (Conflict)?
```bash
# أوقف جميع عمليات البوت
taskkill /F /IM node.exe

# انتظر 5 ثواني
timeout /t 5

# ابدأ البوت
start-bot.bat
```

### البوت بطيء؟
```bash
# أعد تشغيل البوت
taskkill /F /IM node.exe
timeout /t 3
start-bot.bat
```

### خطأ في قاعدة البيانات؟
```bash
# أعد تعيين البوت
reset-bot.bat
```
