# Telegram Bot

## Quick Start
1. Install dependencies: `npm install`
2. Configure your bot token in `config.js`
3. Start the bot: `npm start` or `start-bot.bat`

## Files
- `bot.js` - Main bot logic
- `config.js` - Configuration and messages
- `database.js` - Database operations
- `keyboards.js` - Keyboard layouts
- `bot_database.db` - SQLite database

## Commands
- `/start` - Start the bot and setup language/currency
