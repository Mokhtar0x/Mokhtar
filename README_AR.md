## 🎯 نظرة عامة

بوت تليجرام متكامل يسمح للمستخدمين بالربح من خلال إنشاء حسابات Gmail. البوت يعمل بالعربية فقط ويستخدم الجنيه المصري كعملة وحيدة.

### الميزات الرئيسية:
- ✅ مهام إنشاء Gmail فقط
- ✅ الدفع بالجنيه المصري
- ✅ السحب عبر محفظة كاش
- ✅ لوحة تحكم للأدمن
- ✅ نظام مراجعة الحسابات
- ✅ إحصائيات مفصلة

---

## 💻 المتطلبات

### البرامج المطلوبة:
- Node.js (v14 أو أحدث)
- npm (يأتي مع Node.js)

### المعلومات المطلوبة:
- Bot Token من [@BotFather](https://t.me/BotFather)
- Admin ID (آيدي التليجرام الخاص بك)
- Bot Username (اسم البوت بدون @)

## ⚙️ الإعدادات

### الإعدادات الافتراضية في config.js:

```javascript
// معلومات البوت
BOT_TOKEN: 'توكن_البوت'
ADMIN_ID: 'آيدي_الأدمن'
BOT_USERNAME: 'اسم_البوت'

// المكافآت (بالجنيه المصري)
GMAIL_TASK_REWARD: 10  // مكافأة مهمة الجيميل
MIN_WITHDRAWAL: 50     // الحد الأدنى للسحب

// إعدادات الجيميل
GMAIL_PASSWORD: 'DefaultPass123'  // كلمة المرور الموحدة

// اللغة
DEFAULT_LANGUAGE: 'ar'  // العربية فقط

// العملة
DEFAULT_CURRENCY: 'EGP'  // الجنيه المصري فقط
```

### تغيير الإعدادات:

#### من لوحة الأدمن:
1. افتح البوت
2. اضغط "⚙️ إعدادات النظام"
3. اختر الإعداد المراد تغييره
4. أرسل القيمة الجديدة

#### من ملف config.js:
1. افتح `config.js`
2. عدل القيمة المطلوبة
3. احفظ الملف
4. أعد تشغيل البوت

## 🔄 إعادة التعيين

### متى تحتاج لإعادة التعيين؟
- بدء البوت من الصفر
- حذف جميع البيانات القديمة
- حل مشاكل قاعدة البيانات
- تنظيف البوت بالكامل

### كيفية إعادة التعيين:

#### الطريقة الأولى: ملف reset-bot.bat (موصى بها)
```bash
# قم بتشغيل الملف
reset-bot.bat
```

هذا الملف سيقوم بـ:
1. ✅ إيقاف البوت
2. ✅ إنشاء نسخة احتياطية من قاعدة البيانات
3. ✅ حذف قاعدة البيانات القديمة
4. ✅ مسح السجلات
5. ✅ بدء البوت من جديد

#### الطريقة الثانية: يدوياً
```bash
# 1. إيقاف البوت
taskkill /F /IM node.exe

# 2. نسخ احتياطي (اختياري)
copy bot_database.db backup_bot_database.db

# 3. حذف قاعدة البيانات
del bot_database.db
del bot_database.db-shm
del bot_database.db-wal

# 4. مسح السجلات (اختياري)
del logs\errors.log
del logs\activity.log

# 5. بدء البوت
node bot.js
```

## ❓ الأسئلة الشائعة

### س: البوت لا يرد على الرسائل؟
**ج:** تأكد من:
- البوت يعمل (تحقق من نافذة الأوامر)
- التوكن صحيح في config.js
- لا يوجد بوت آخر يعمل بنفس التوكن

### س: كيف أحصل على Bot Token؟
**ج:** 
1. افتح [@BotFather](https://t.me/BotFather)
2. أرسل `/newbot`
3. اتبع التعليمات
4. انسخ التوكن

### س: كيف أحصل على Admin ID؟
**ج:**
1. افتح [@userinfobot](https://t.me/userinfobot)
2. أرسل `/start`
3. انسخ الآيدي

### س: البوت يتوقف عن العمل؟
**ج:** تحقق من:
- ملف `logs/errors.log` للأخطاء
- اتصال الإنترنت
- مساحة القرص الصلب

### س: كيف أغير مكافأة المهمة؟
**ج:**
1. افتح البوت كأدمن
2. اضغط "⚙️ إعدادات النظام"
3. اضغط "💰 إعدادات المكافآت"
4. اضغط "📱 مكافأة مهمة الجيميل"
5. أرسل المكافأة الجديدة

### س: كيف أغير الحد الأدنى للسحب؟
**ج:**
1. افتح البوت كأدمن
2. اضغط "⚙️ إعدادات النظام"
3. اضغط "💳 تغيير الحد الأدنى للسحب"
4. أرسل الحد الجديد

### س: كيف أعدل رسالة الدعم؟
**ج:**
1. افتح البوت كأدمن
2. اضغط "⚙️ إعدادات النظام"
3. اضغط "💬 تعديل رسالة الدعم"
4. أرسل الرسالة الجديدة

### س: كيف أفعل/أعطل مهام الجيميل؟
**ج:**
1. افتح البوت كأدمن
2. اضغط "🎮 التحكم في المهام"
3. اضغط "📱 مهمة إنشاء الجيميل"

### س: كيف أصدر جميع الإيميلات؟
**ج:**
1. افتح البوت كأدمن
2. اضغط "📥 إدارة الإيميلات الجماعية"
3. اضغط "📤 تصدير كل الإيميلات"
4. سيرسل لك البوت ملف txt

### س: كيف أقبل/أرفض عدة حسابات دفعة واحدة؟
**ج:**
1. افتح البوت كأدمن
2. اضغط "📥 إدارة الإيميلات الجماعية"
3. اختر:
   - "✅ إرسال المقبولة وقبولها" للقبول
   - "❌ إرسال المرفوضة ورفضها" للرفض
4. أرسل الإيميلات (كل إيميل في سطر)

## 🐛 حل المشاكل

### المشكلة: البوت لا يبدأ
**الحل:**
```bash
# تحقق من الأخطاء
node bot.js

# إذا ظهر خطأ في المكتبات
npm install

# إذا ظهر خطأ في قاعدة البيانات
del bot_database.db
node bot.js
```

### المشكلة: خطأ 409 (Conflict)
**الحل:**
```bash
# أوقف جميع عمليات البوت
taskkill /F /IM node.exe

# انتظر 5 ثواني
timeout /t 5

# ابدأ البوت مرة أخرى
node bot.js
```

### المشكلة: البوت بطيء
**الحل:**
1. تحقق من اتصال الإنترنت
2. أعد تشغيل البوت
3. امسح السجلات القديمة
4. أعد تشغيل الكمبيوتر
